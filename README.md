## 📦 What is this repository?

This is my **`.github`** special repository — the **community & automation hub** for all my projects. It provides:

- 📜 **Default community health files** (`CONTRIBUTING`, `CODE_OF_CONDUCT`, `SECURITY`, `SUPPORT`) that automatically apply to **every repo** I own that doesn't override them.
- 🐛 **Issue templates** & **PR template** for consistent, well-structured contributions.
- 🤖 **GitHub Actions** — Dependabot auto-merge, stale-issue management, and a welcome bot for first-time contributors.
